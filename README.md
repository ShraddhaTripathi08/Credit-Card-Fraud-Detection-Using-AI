# Credit Card Fraud Detection Using Artificial Intelligence

This project focuses on detecting fraudulent credit card transactions using Artificial Intelligence and Machine Learning techniques.

## Objective
To compare multiple AI models and identify the most effective model for credit card fraud detection.

## Models Used
- Logistic Regression
- Decision Tree
- Random Forest
- XGBoost
- Neural Networks
- Isolation Forest

## Dataset
European Credit Card Fraud Dataset (Kaggle)

## Tools & Technologies
- Python
- Machine Learning
- AI Algorithms
- Data Analysis

## Outcome
XGBoost and Random Forest showed the best performance in detecting fraudulent transactions.
